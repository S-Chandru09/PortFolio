# HTML Concepts
- Paired Tags
- Unpaired Tags(Self Closing tags)
- Blocked level Tags
- Inliine Tags
- Semantics Tags
- Non Semantic 

1.**Paired Tags:**
    Tags that are need to be closed properly after opening
    ```Eg: <h1></h1>,<p></p>,<div></div>,<html></html>```
2.**Unpaired Tags:**
    ```eg:<!-- br,hr,img,input -->```
3.**Blocked level Tags:**
    block level execution
    ```eg:<div>,<p>,<ul>,<ol>,<section>```
4.**Inline Tags:**
    tages used within a line
    ```eg:<img>,<a>,<b>,<i>```
5.**Semantic tages:**
    tags which give meaning for the content
    ```eg:<header>,<nav>,<section>,<footer>,```
6.**Non Semantic:**
    ```eg:<div>,<span>```
